# DigiTools

DigiTools is a modern digital tools marketplace UI built with React. It includes a product catalog, cart toggling, pricing plans, onboarding steps, and a responsive marketing layout inspired by Figma.

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- React-Toastify
- JSON data source

## Key Features

- Product and cart view toggle with dynamic cart count in the navbar.
- Add to cart, remove from cart, and proceed-to-checkout flow with toast notifications.
- Responsive UI sections for hero, stats, products, steps, pricing, CTA, and footer.

## Functional Highlights

- Product cards are generated from JSON data (9 products).
- Cart supports per-item remove and full checkout reset.
- "Buy Now" button changes to "Added to Cart" for selected product.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
